# easyou-node

### 基于 thinkjs【1.2.10】 的博客系统 

## 特点

* 急速
* 高效
* 简洁

## 运行环境

* node > 0.12.0
* MySQL

## 使用方法

1.克隆必要文件

```
git clone https://github.com/eyblog/easyou-node.git
```

2.导入相关数据

导入根目录下的`ey.sql`到 MySQL 数据库，配置文件见[./App/Conf/config.js](./App/Conf/config.js)

3.安装必要依赖+启动项目

```shell
npm install && npm start
```
